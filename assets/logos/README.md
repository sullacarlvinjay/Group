# Logos Folder

This folder contains logo images for the website.

## Required Logo Files:
- mushguard.png (or .svg)
- prophane.png (or .svg)
- sakay.png (or .svg)

## Recommended Specifications:
- Format: PNG with transparent background or SVG
- Size: 200x200 pixels (will be scaled to 80x80)
- Style: White or light colors for best visibility on dark background

## Links:
- MushGuard: https://mushguard.onrender.com
- Prophane: https://prophane.onrender.com/
- Sakay: https://sakay-1-cwdp.onrender.com/

Place your logo files in this folder and the HTML will automatically pick them up.
